# open_voicebank_readme

使用中の音源のreadme.txtを開くUTAUプラグイン

## インストール

open_voicebank_readme.zip をダウンロードして、D&Dでインストールしてください。

## 仕様

使用中の音源フォルダにあるreadme.txtを開きます。

## 開発環境

- Windows 10
- Python 3.9.5
- utaupy 1.11.3 のあとの開発版
- UTAU 0.4.18e
